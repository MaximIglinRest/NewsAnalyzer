### Установка с помощью окружения poetry
1. ``poetry shell``

2. ``poetry install`` 

### Установка с помощью окружения python-venv
1. ``python<your version> -m venv env``

2. ``. ./env/bin/activate``

3. ``pip install -r requirements.txt``

### Запуск:
`` uvicorn server:app --reload --port 8000``
